# Plataforma-educativa-games

Plataforma web de jogos educativos com foco pedagogico, mecanicas ludicas e acompanhamento de progresso para criancas e adolescentes.

## Objetivo
Criar uma plataforma inspirada em portais de jogos, mas com foco em aprendizagem, engajamento e acompanhamento pedagogico.
